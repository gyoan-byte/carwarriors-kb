# Difficult Conversations
## Agent De-escalation Layer

### Purpose
Define runtime handling of hostile, skeptical, or stalled conversations.

### Runtime Policy
- Validate briefly, then request one concrete next step.
- Keep neutral and non-defensive language.
- Apply strike-based closure for abuse.

### Compliance Dependencies
- Safety and data handling follow `knowledge_base/07_Safety_Compliance.md`.
- Escalation timing follows `knowledge_base/09_Escalation_Policy.md`.
