# Transitions
## Agent Handoff Language Layer

### Purpose
Provide execution patterns for moving from qualification to advisor handoff.

### Transition Sequence
1. Confirm intent.
2. State that exact confirmation is advisor-owned.
3. Request one next action.

### Execution Constraints
- One question per turn.
- No refusal-only phrasing.
- Channel usage follows `knowledge_base/07_Safety_Compliance.md`.
