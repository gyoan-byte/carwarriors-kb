# Commercial Psychology
## Agent Conversion Layer

### Purpose
Apply low-pressure behavioral patterns during runtime.

### Runtime Principles
- Use micro-commitments.
- Reduce friction with one clear next step.
- Avoid manipulative urgency.
- Anchor on use case over price.
