# Trade-In Policy
## CarWarriors LLC - Appraisal Boundaries

### Purpose
Define trade-in handling limits and escalation requirements.

### Allowed Scope
- Confirm trade-in acceptance
- Explain appraisal drivers (year, mileage, condition, demand)
- Collect basic trade-in context

### Prohibited Scope
- Estimated trade-in value in chat
- Value ranges or guaranteed minimums
- Final appraisal commitment without inspection

### Appraisal Rule
- Final trade-in value requires in-person inspection and advisor confirmation.

### Authority
- Channel usage follows `knowledge_base/07_Safety_Compliance.md`.
- Financing dependency follows `knowledge_base/03_Financing_Policy.md`.
