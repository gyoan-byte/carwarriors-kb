# BASE DE CONOCIMIENTO - TRADE-IN
## CarWarriors LLC - Política de Intercambio

### **INFORMACIÓN AUTORIZADA**

El bot puede usar:
- "Se aceptan intercambios"
- "Valor sujeto a inspección"
- "Se evalúa año, millaje, condición"
- "Claro, aceptamos trade-in"

### **INFORMACIÓN PROHIBIDA**

El bot nunca debe:
- Dar valor estimado
- Dar rangos aproximados
- Prometer valor mínimo
- Confirmar aceptación sin inspección

### **FLUJO DE TRADE-IN**

1. **Detección:** Usuario menciona vehículo actual
2. **Confirmación:** "Sí, aceptamos trade-in"
3. **Información básica:** "¿Qué vehículo tienes para intercambiar?"
4. **Escalación:** Transferir a asesor para evaluación

### **RESPUESTAS TIPO**

**Consulta inicial:**
- "Claro, aceptamos trade-in. ¿Qué vehículo tienes?"
- "Sí, podemos evaluar tu vehículo actual. ¿Qué modelo es?"

**Sobre valor:**
- "El valor se determina tras inspección"
- "Depende de año, millaje y condición"

**Sobre proceso:**
- "Lo evaluamos cuando visitas el dealer"
- "Te damos valor final tras revisión física"

### **FACTORES DE EVALUACIÓN**

- Año del vehículo
- Millaje actual
- Condición mecánica y estética
- Historial de mantenimiento
- Demanda del modelo

### **ESCALACIÓN OBLIGATORIA**

Siempre escalar a asesor para:
- Valoraciones específicas
- Aceptación condicional
- Negociaciones de valor
